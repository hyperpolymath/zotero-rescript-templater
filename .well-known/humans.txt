/* TECHNOLOGY COLOPHON */

Languages:
- PowerShell 7+ (Windows/Linux/macOS scaffolding)
- Racket 8.0+ (homoiconic scaffolding)
- Bash 4+ (Unix-like systems)
- C# (embedded XXHash64 implementation)
- JSON (template definitions)
- YAML (CI/CD workflows)

Template Languages:
- ReScript (practitioner template)
- TypeScript (student template)
- JavaScript (all templates)
- XUL (Zotero UI overlays)

Build Tools:
- GitHub Actions (CI/CD)
- Pester (PowerShell testing)
- rackunit (Racket testing)
- Podman/Docker (containerization)
- npm (JavaScript package management)

Testing:
- 70+ automated tests
- Cross-platform validation
- Integrity verification (XXHash64/SHA256)
- Static analysis (PSScriptAnalyzer, racket check)
- Security scanning (CodeQL, Trivy)

Infrastructure:
- Version Control: Git
- Hosting: GitHub
- CI/CD: GitHub Actions
- Container Registry: GitHub Container Registry
- Documentation: Markdown (GitHub Flavored)
- Issue Tracking: GitHub Issues
- Discussions: GitHub Discussions

Standards Compliance:
